## ✨ Features
- Modern SaaS-style design with glassmorphism
- Dark/Light mode toggle with localStorage persistence
- Smooth scroll animations and reveal effects
- Typewriter hero effect
- Responsive design (mobile-first)
- Custom cursor effects (desktop)
- Loading screen with progress bar
- Project showcase with case studies
- Interactive timeline
- Contact form

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Tailwind CSS (CDN)
- Font Awesome
- Google Fonts (Inter)

## 🚀 Getting Started
1. Clone the repo
2. Open `index.html` in browser
3. No build tools required!

## 📱 Pages
- **Homepage** (`index.html`) - Overview of experience, skills, and featured work
- **Projects Hub** (`projects.html`) - Filterable gallery of all projects
- **Case Studies** (`pages/*.html`) - Deep dives into individual projects
